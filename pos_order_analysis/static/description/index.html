Description

This module allows us to analyse orders of point of sale. It will provide following features:

Print/Download Features :

  • Printing the derived chart.
  • Download as PDF Document.
  • Download as PNG Image.
  • Download as JPEG Image.
  • Download as SVG Vector Image.

Filter Options:

  • Analyse orders related to date.
  • Analyse orders related to customer.
  • Analyse orders related to Product(s).

Zooming Features.

POS Orders Analysis Chart

This module draws the chart as the combination of pie chart and columns charts. Based on the filters, the orders will be displayed in the chart. Column chart will show the product consumed on each pos order and in pie chart one can see the total consumption.

User can filter orders based on specified dates. Orders between those dates will we display in chart.

Print/Download Chart

User can download chart in different formats like PNG, JPEG, SVG and PDF.

Filtering Orders Related To Customer

User can also filter pos order based on the customer, so only those orders will be displayed in which particular customer is set.

Filtering Orders Related To Product

User can also filter the product. One can select multiple products as you can see in following image.

Filtering Orders Related To Product

This module uses pie & column chart from the well-known Charting Library HighCharts.
