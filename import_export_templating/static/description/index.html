Import / Export Templating

This module allows you to Export / Import any data of model. You just need to Download Blank Template based on type selection and To Import select type and load downloded file with data.

Importing bulk data into Odoo-compatible format has been a headache for end users and implementers. This module allows you to fetch a template first from odoo, you may feed the data accordingly and import back. You may here choose various options whether ou want to create master data or skip. This module helps you to import any kind of data by creation / modification.

Export

  • Export blank template based on type selection of documents.
  • Adding selective fields click on add an item for template.
  • Using required button you can select required fields only!
  • Using select all button you can select all fields of document.
  • Using uncheck button you can clear all selective fields for blank template.

Import

  • Feature to Import data based on Downloded Template.
  • Select document type and load related tempalte.
  • You can create Many2one and Many2many records!
  • Validate the Xls. Data Provided
