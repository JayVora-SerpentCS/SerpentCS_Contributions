Web one2many Kanban

This module is developed to extend the functionality of kanban view. There are certain occasions where you need to display one2many records in kanban view. This module fills the gap at certain extent by allowing to display one2many records in kanban view.

This is the one2many records which is displayed in form view.

You need to define o2m field in kanban view definition and use for loop(same way we use in qweb) to display fields in kanban view.
